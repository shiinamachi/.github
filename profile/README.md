Whenever we build something worth sharing, we release it publicly and actively maintain it.

Lately, we've been really interested in:

- Minecraft
- Bare-metal Server Operation
- Customizing for Windows OS

## Projects

### [hyper-pwt](https://github.com/shiinamachi/hyper-pwt)

hyper-pwt is a command-line tool for building widgets. It is designed to be a modern replacement for the old Mendix Pluggable Widget Tools. You can easily switch to this tool without needing to change your existing setup.

### [wintender](https://github.com/shiinamachi/wintender)

Wintender is a Zebar widget designed to bring the functionality and convenient top bar of macOS's Bartender to Windows via Zebar.

### [codex-gpt-pro-plugin](https://github.com/shiinamachi/codex-gpt-pro-plugin)

codex-gpt-pro-plugin is a plugin developed to allow the use of ChatGPT's GPT Pro model within the Codex app and CLI.

### [interview-tutor](https://github.com/shiinamachi/interview-tutor)

interview-tutor is a Local First web app developed to help you prepare for interviews with the support of LLM models.
